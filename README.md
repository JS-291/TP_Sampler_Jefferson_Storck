# TP_Sampler_Jefferson_Storck

## RestServer
- RestServer is the folder of the REST server. 
- It's a modified version of the server from seance 2. 
- To run use "npm run start".

## Main
- Main is the folder of the audio sampler.
- The audio presets are fetched from the REST server, It must be running.
- To run use "npm run start"
